✨ Características Principales

Todo lo que necesitas para analizar tus datos de ventas en un solo lugar

Análisis Univariado

Distribuciones, estadísticas descriptivas, correlaciones y análisis de variables individuales con gráficos interactivos.

Análisis Bivariado

Relaciones entre variables, tendencias temporales, análisis geográfico y comparaciones cruzadas.

Análisis Predictivo

Modelos de regresión lineal, clasificación y predicción de tendencias futuras con machine learning.

Filtros Interactivos

Filtra por año, país, producto, estado, territorio y rango de ventas para análisis personalizados.

KPIs Dinámicos

Métricas clave con comparación temporal, tendencias y alertas visuales automáticas.

Responsive Design

Interfaz adaptable que funciona perfectamente en desktop, tablet y dispositivos móviles.

📊 Tipos de Análisis

Explora diferentes niveles de análisis para tus datos de ventas

Análisis Univariado

Distribución de Ventas

Histogramas y boxplots para entender la distribución de ventas y cantidades.

Estadísticas Descriptivas

Media, mediana, moda, desviación estándar, percentiles y más.

Matriz de Correlación

Heatmap de correlación entre variables numéricas del dataset.

Variables Categóricas

Distribución de países, productos, estados y territorios.

Análisis Bivariado

Tendencias Temporales

Evolución de ventas por año, mes y trimestre con líneas de tendencia.

Análisis Geográfico

Comparación de ventas por país y territorio con visualizaciones.

Por Producto

Ventas y cantidades por línea de producto y tamaño de trato.

Scatter Plots

Relaciones entre cantidad, precio y ventas con burbujas de color.

Análisis Predictivo

Regresión Lineal

Predicción de ventas futuras con métricas de evaluación del modelo.

Clasificación

Predicción del tamaño de trato (Small/Medium/Large) con ML.

Demo Interactiva

Prueba el modelo predictivo con parámetros personalizados.

Detección de Tendencias

Identificación de patrones estacionales y tendencias de mercado.
