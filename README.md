> 🚀 Tablero de control interactivo para análisis descriptivo (univariado y bivariado) y predictivo de datos de ventas. Desarrollado con Python, Streamlit y Plotly.

## 📋 Características Principales
- 📈 **KPIs Dinámicos**: Ventas totales, pedidos, ticket promedio y tasa de éxito con variación temporal.
- 📊 **Análisis Univariado**: Distribuciones, estadísticas descriptivas, matriz de correlación y gráficos de variables categóricas.
- 🔗 **Análisis Bivariado**: Tendencias temporales, ventas por categoría, scatter plots interactivos y heatmaps.
- 🔮 **Análisis Predictivo**: Regresión lineal para proyección de ventas, clasificación de tamaño de trato (ML) y demo interactiva.
- 🎛️ **Filtros Avanzados**: Filtrado por año, país, línea de producto, estado, territorio y rango de ventas.
- 📱 **Diseño Responsivo**: Interfaz moderna y adaptable a cualquier dispositivo.
